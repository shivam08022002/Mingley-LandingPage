# Mingley - Dating App Landing Page

Mingley is a modern, responsive landing page for a dating application. It's designed to showcase features like instant registration, chat, and meeting new people, all in a stylish and user-friendly interface.

## ✨ Features

- **Hero Section**: Engaging introduction with a clear call-to-action for app downloads.
- **Interactive Chat Previews**: Visualizing user interactions directly on the landing page.
- **Features Showcase**: Highlighting key benefits of the platform.
- **App Screens**: Previews of the actual application interface.
- **Testimonials**: Social proof from users of the app.
- **Download Section**: Easy access to Google Play and Apple App Store links.
- **Responsive Design**: Optimized for all devices and screen sizes.
- **Smooth Animations**: Scroll-triggered fade-up effects for an enhanced user experience.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Styling**: CSS3
- **Animations**: Intersection Observer API for scroll-triggered effects.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Mingley-LandingPage.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Mingley-LandingPage
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

To start the development server:
```bash
npm run dev
```

### Build

To build the project for production:
```bash
npm run build
```
